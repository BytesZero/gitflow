# gitflow
SourceTree Git Flow 介绍

## 什么是 Git Flow？
- 一系列 Git 命令的组合
- 建立一种规范，为了让 Git 在协作使用时更加顺畅
## 分支介绍

- `master`  主分支
- `develop` 开发分支
- `future/*` 功能分支
- `release` 预发分支
- `hotfix/*` 修复分支
- 其他辅助、临时分支
